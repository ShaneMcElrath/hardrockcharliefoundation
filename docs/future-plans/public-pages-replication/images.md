# Content Images Catalog

> **Instructions:** This file lists every image that needs to be saved into the repo for the
> site to display correctly. These are NOT the design screenshots — these are the actual photos,
> logos, and graphics that appear on the website pages.
>
> **How to use:**
> 1. Look for `<!-- ✏️ USER EDIT -->` markers — confirm or update each entry
> 2. Save each image to the listed path with the listed filename
> 3. Check off each image as you save it
>
> **Where to save images:** `static/images/` in the project root. The folder structure is
> listed below.

## Folder Structure

Create this structure inside `static/images/`:

```
static/images/
```
